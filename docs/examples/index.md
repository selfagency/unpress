# Examples

These examples show practical ways to run Unpress with `pnpx` or `npx`, `.env`, and optional config files.

## Available examples

- [Simple blog](./simple-blog/)
- [Multi-author blog](./multi-author/)
- [S3 media reupload](./s3-media/)
- [Archival migration](./archival/)

Each example includes:

- a sample `.env`
- a sample `unpress.yml` when useful
- the command to run
- what output to expect
